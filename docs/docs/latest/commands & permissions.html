Commands & Permissions

Commands


/rm
Permission: recipemanager.command.rm
Aliases: recipemanager, rmhelp		plugin info and available commands


/rmrecipes <material>:[data]:[amount] [i|r]
Permission: recipemanager.command.rmrecipes
Aliases: recipe, recipes		find recipes that use the specified item


/rmnext
Permission: recipemanager.command.rmnext
Aliases: N/A		go to the next page in an /rmrecipes query


/rmprev
Permission: recipemanager.command.rmprev
Aliases: N/A		go the previous page in an /rmrecipes query


/rmfinditem <name>
Permission: recipemanager.command.rmfinditem
Aliases: finditem		search for items by name


/rmcheck
Permission: recipemanager.command.rmcheck
Aliases: checkrecipes		simulates loading recipe files to check for errors


/rmreload
Permission: recipemanager.command.rmreload
Aliases: reloadrecipes		reload recipes/settings/books/etc.


/rmreloadbooks
Permission: recipemanager.command.rmreloadbooks
Aliases: reloadbooks		reload only recipe books


/rmextract [special]
Permission: recipemanager.command.rmextract
Aliases: rmimport, importrecipes, extractrecipes		makes all recipes from other plugins or mods into a text file to allow overriding and editing


/rmgetbook <title> [#<volume>]
Permission: recipemanager.command.rmgetbook
Aliases: getrecipebook		gives a recipe book


/rmbooks
Permission: recipemanager.command.rmbooks
Aliases: recipebooks		lists available recipe books


/rmupdate
Permission: recipemanager.command.rmupdate
Aliases: rmcheckupdates		checks if there's a newer version available for the plugin


/rmcreaterecipe
Permission: recipemanager.command.rmcreaterecipe
Aliases: createrecipe		creates a recipe from the leftmost 3x3 ingredients in an inventory and the held item as a result


/rmdebug
Permission: recipemanager.command.rmdebug
Aliases: N/A		debugging help for checking permissions and block data

For the flag permissions you can use the flag's aliases as well, I filtered them from this list because it would've become too long, but the permissions are there.
For example, recipemanager.flag.modexp and recipemanager.flag.xp both affect the same flag, the @modexp flag, since 'xp' is an alias for 'modexp'.
